# InsEmoji.vim
Insert Emoji to Vim

## Installation
Vundle:

```
Plugin 'laddge/InsEmoji.vim'
```

According to the plugin manager you use.

## Usage
### 1. Insert by name

```
:InsEmoji <name>
```

\<name\> is defined in [emoji/emoji.json](emoji/emoji.json)

ex:

```
:InsEmoji SUSHI " --> üç£
```

### 2. Insert from menu

```
:InsEmojiMenu
```

You can search like any other buffer.
